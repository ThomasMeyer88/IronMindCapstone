Update Program

After you create a program, you are able to
make any adjustments for them. That includes adding, editing,
and removing any exercises for that active program.



Log Workout

You are able to log each of the workouts within the
program. This will allow you to keep track of your daily
routine. You can now proceed to the "View Progress"
section for detailed progress.

`

View Progress

Users are able to see a graph that will
show volume, intensity or both for each workout you have
completed. This is a great way to see that you are moving
forward to you ultimate goal!


Coaches

You are able to search for a professional coach that can
guide you to be the best you can be.From here
you will be able to see the coach and a
description of themselves.
